# Live-Electronics Tutorial Translation

#### 介绍

本仓库为Live-Electronics Tutorial的中文翻译企划。

本教程是由Alexandre Torres Porres编写的，内置在ELSE中的一款教程，体系庞大、内容海量，基本解释了Pd的一切。

本翻译基于Plugdata，与Pure Data原生环境有所差异，译文亦会针对此作出说明。

#### 原仓库

[Pure Data](https://github.com/pure-data)

[Plugdata](https://github.com/plugdata-team/plugdata)

[ELSE](https://github.com/porres/pd-else)

[Live-Electronics Tutorial](https://github.com/porres/pd-else/tree/master/Documentation/Live-Electronics-Tutorial)

#### 使用说明

1.  本教程所有内容均基于Live-Electronics Tutorial和Pure Data、Plugdata，使用.pd格式分发，你需要使用Pure Data环境以打开这些文件。
(download Pd from: http://msp.ucsd.edu/software.html)

    Plugdata官网：https://plugdata.org/index.html (内置Pure Data Vanilla与ELSE、Cyclone库，本教程基于此软件进行一定表述上的更改)

2.    由于Plugdata的comment注释不支持中文句号，英文句号在划分中文文字时又不够显眼，故使用“//”进行段落划分。必要时使用[else/note]对象。

3.    本教程虽然从结构上来说包含了面向完全新手的部分，但碍于篇幅原因，对很多的基础概念没能解释清楚，如在阅读过程中发现有先前未解释的内容，请先试着使用Pd内部文档或者网上的资料自行解决，并提出Issue。我会补充相应的部分。

4.   本教程翻译难度较大，Alice Collection不保证随时更新至最新版本。
    版本更新时间：2026.7.24
